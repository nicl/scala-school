Scala School
============

Learning Scala at the Guardian.

## Getting started

Enter sbt:

    $ ./sbt

Then, to build:

    $ gen-idea // if using Intellij

To run:

    $ run

To test:

    $ test

To enter the REPL:

    $ console

## Topics

### Week 1

* using the REPL
* basic types
* vals + vars
* method calls and operators
* scaladoc

### Week 2

* methods and functions
* collections
* map, filter, etc.

### Week 3

* classes
* objects
* case classes
* companion objects
